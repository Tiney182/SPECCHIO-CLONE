Note. If you are upgrading an existing installation of SPECCHIO, ensure that you keep the file db_config.txt from your existing installation. This file contains your existing account configuration. If you install the new version of SPECCHIO into the same folder as your existing installation, the installer will first prompt you to overwrite this folder. Answer "Yes". It will then prompt you overwrite db_config.txt. Answer "No".
